## Cómo transforma los datos en una ruta de acción más ordenada

Estas vistas pueden ordenarse por línea de negocio, fase del flujo, grupo de clientes, producto o cualquier otra dimensión útil.

Una vez creadas esas vistas, **Profitiva Fundres** permite añadir **reglas declarativas**. Esas reglas indican qué condiciones deben interpretarse como relevantes. Un pequeño cambio puede ser normal. Cuando estas condiciones están definidas por adelantado, el equipo pierde menos tiempo discutiendo qué significa lo que acaba de aparecer.

Si una regla se activa, la plataforma puede señalar el caso, sugerir un siguiente paso o colocarlo dentro de una secuencia de revisión.
